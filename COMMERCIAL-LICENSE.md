# Commercial licensing

> **Status: placeholder.** This describes intent, not an enforceable contract. See the disclaimer at the bottom before relying on anything here.

## Why this exists

This project's code is licensed under **AGPL-3.0** (see `LICENSE`). AGPL requires that anyone who modifies this code and offers it as a network service (SaaS or otherwise) publish their modified source to the users of that service. That's by design — it's meant to keep improvements to this project public rather than disappearing into a closed competitor.

If your organization cannot accept that obligation — for example, your legal/compliance policy bans AGPL dependencies, or you want to build a closed-source product/service on top of this code without publishing your modifications — a separate commercial license, with different terms, is the intended path.

## Current state

- No commercial license contract exists yet — because there has been no commercial inquiry to write one for.
- Terms (price, scope, support, duration) are not defined and shouldn't be assumed from this file.
- The real contract will be drafted with legal review the first time there's an actual inquiry, not speculatively.
